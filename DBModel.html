Normalizes a schema
(column: String -> column: {type: String})

DBModel.addValidator(validator)

Adds a validator

A validator must return false(boolean) or error message(string), or throw an Error exception if invalid

Parameters:

  • {Function} validator
    • {DBModel} record

DBModel.belongsTo(target_model, [options])

Adds a belongs-to association

Parameters:

  • {Class} target_model
  • {Object} options
    • {String} as
    • {String} foreign_key

DBModel.build([data={}])

Creates a record.
'Model.build(data)' is the same as 'new Model(data)'

Parameters:

  • {Object} data={}

Returns:

  • {DBModel}

DBModel::constructor([data={}])

Creates a record

Parameters:

  • {Object} data={}

DBModel.count([condition], [callback])

Counts records by conditions

Parameters:

  • {Object} condition
  • {Function} callback
    • {Error} error
    • {Number} count

Returns:

  • {DBQuery}

DBModel.create([data={}], callback)

Creates a record and saves it to the database
'Model.create(data, callback)' is the same as 'Model.build(data).save(callback)'

Parameters:

  • {Object} data={}
  • {Function} callback
    • {Error} error
    • {DBModel} record created record

DBModel.delete([condition], [callback])
